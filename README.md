Утилита командной строки page-loader, которая скачивает страницу из сети и кладёт в указанную директорию (по умолчанию в директорию запуска программы).
Программа возвращает полный путь к загруженному файлу.
Использование: команда page-loader и 2 аргумента. Первый аргумент путь для сохранения файлов, второй адрес вэб сайта. По умолчанию сохраняет в директорию запуска.
